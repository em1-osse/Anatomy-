# منصة التشريح التعليمية | Anatomy Learning Platform

منصة تعليمية تفاعلية لعمل ملخصات ودروس جامعية، مع دعم العربية والإنجليزية، واختبارات تفاعلية، ونطق صحيح، وخاصية الطباعة.

## 📁 الملفات

- `index.html` — الهيكل الأساسي للصفحة
- `styles.css` — التصميم والألوان والتجاوب مع الشاشات
- `data.js` — بيانات الدورات والمحاضرات (هنا تضيف دروسك الجديدة)
- `app.js` — المنطق البرمجي للتطبيق (تصفح، بحث، اختبارات، نطق...)
- `README.md` — هذا الملف (دليل الاستخدام)

## 🚀 كيفية الاستخدام

### 1. فتح الموقع محلياً

افتح الملف `index.html` في أي متصفح حديث. لا تحتاج إلى خادم (Server) للتشغيل.

أو يمكنك استخدام خادم محلي بسيط:
```bash
cd anatomy-platform
python3 -m http.server 8000
```
ثم افتح: `http://localhost:8000`

### 2. إضافة دورة جديدة

افتح ملف `data.js` وابحث عن قسم `courses`. انسخ كائن الدورة `template-course` والصقه في النهاية، ثم عدّل:

- `id`: معرف فريد للدورة (مثل: `"biology-week1"`)
- `title`: `{ ar: "...", en: "..." }`
- `subtitle`: `{ ar: "...", en: "..." }`
- `description`: نبذة عن الدورة
- `image`: رابط صورة الغلاف
- `badge`: تصنيف الدورة
- `lectures`: مصفوفة المحاضرات

### 3. إضافة محاضرة جديدة

كل محاضرة تحتوي على:

```javascript
{
    id: "lecture-id",
    icon: "📚",
    title: { ar: "عنوان المحاضرة", en: "Lecture Title" },
    topics: [
        { ar: "موضوع 1", en: "Topic 1" },
        { ar: "موضوع 2", en: "Topic 2" }
    ],
    sections: [...],
    quiz: [...]
}
```

### 4. أنواع الأقسام داخل المحاضرة

يمكنك استخدام هذه الأنواع في `sections`:

- `intro` — مقدمة نصية
- `text` — نص عادي
- `images` — معرض صور مع تعليقات
- `terms` — قائمة مصطلحات مع زر نطق
- `table` — جدول بيانات
- `bones` — بطاقات عظام/مصطلحات تفصيلية

### 5. إضافة اختبار

```javascript
quiz: [
    {
        question: { ar: "السؤال؟", en: "Question?" },
        options: [
            { ar: "خيار 1", en: "Option 1" },
            { ar: "خيار 2", en: "Option 2" },
            { ar: "خيار 3", en: "Option 3" },
            { ar: "خيار 4", en: "Option 4" }
        ],
        correct: 0  // رقم الخيار الصحيح (0, 1, 2, 3)
    }
]
```

## 🔧 تخصيص التصميم

يمكنك تغيير الألوان في ملف `styles.css` من القسم `:root` في الأعلى:

```css
:root {
    --primary-color: #2563eb;      /* اللون الرئيسي */
    --secondary-color: #06b6d4;    /* اللون الثانوي */
    --accent-color: #f59e0b;       /* لون التمييز */
    --success-color: #10b981;      /* لون النجاح */
    --danger-color: #ef4444;       /* لون الخطأ */
}
```

## 💡 نصائح

- استخدم صور من مواقع مجانية مثل [Pexels](https://pexels.com) أو [Unsplash](https://unsplash.com).
- لإضافة صور خاصة بك، ضعها في مجلد `images/` ثم استخدم `images/photo.jpg` كرابط.
- النصوص الطويلة جداً يمكن تقسيمها إلى عدة `sections`.
- جرب المنصة محلياً قبل رفعها على الإنترنت.

## 📞 دعم

إذا واجهت مشكلة، تأكد من:
1. أن جميع الملفات الأربعة في نفس المجلد.
2. أن المتصفح محدث (Chrome, Firefox, Edge, Safari).
3. أن `data.js` لا يحتوي على أخطاء في الفواصل أو الأقواس.
